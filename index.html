wopowrimo-2011 by alaricsp

These are my WoPoWriMo efforts for 2011. This month is a bit busy for me personally, so I'm not going to commit to doing something every day. This year, I'm going to write in Lojban.

I'm editing them in Emacs on a NetBSD laptop, version-tracking it in git, and publishing it on GitHub via their pages system. I couldn't be more "Bay Area Trendy Geek" unless I was doing it from my iPhone, and had an oriental-looking girlfriend.

Lojban is a language based around sometimes radically different concepts to English. Therefore, I am presenting the poems in Lojban, then as a literal English translation, then a proper English translation (where possible).

The Poems

  • lo namcu (The set of things which are a number / The Numbers)
